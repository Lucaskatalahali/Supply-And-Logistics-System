# Supply and Logistics System

A web-based Supply and Logistics Management System developed using ASP.NET Core MVC and PostgreSQL.  
The system is designed to manage supplies, inventory, and logistics operations in an organized and efficient way.

## Features
- Manage supply and inventory records (CRUD operations)
- Create, edit, and delete supply items
- Organized MVC architecture (Model–View–Controller)
- Database integration with PostgreSQL

## Technologies Used
- C#
- ASP.NET Core MVC
- PostgreSQL
- Entity Framework Core
